# 1 需求分析

在课程计划维护界面点击下一步进入师资管理界面

![](https://cyan-images.oss-cn-shanghai.aliyuncs.com/images/online-education-20230122-45.png)

点击添加教师打开添加界面 ，不用实现上传照片

> 注意

只允许向机构自己的课程中添加教师、删除教师，机构id统一使用1232141425L

# 2 接口定义

> 查询教师接口

```json
GET /courseTeacher/list/75
75为课程id，请求参数为课程id
```

> 添加教师接口

```json
POST /courseTeacher
```
