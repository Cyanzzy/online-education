# 0 项目简介

本项目以在线教育业务为基础，在线教育网站包括用户端、机构端、运营端。采用前后端分离架构，基于Spring Cloud 微服务技术栈进行设计，采用Nginx、SpringBoot、Spring Cloud、MyBatis-Plus、MQ、Redis、Elasticsearch等框架和中间件为基础进行开发
